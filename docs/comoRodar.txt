================================================
 GUIA DE EXECUÇÃO - A LENDA DE LINK (BUSCA A*)
================================================

Projeto desenvolvido por Otávio Guimarães e Arthur Campos.
Repositório GitHub: https://github.com/otavioagf/Trabalho-IA-Zelda-Pathfinding

Este guia descreve os passos para configurar o ambiente e executar o programa em um novo dispositivo a partir do arquivo ZIP.

### Passo 1: Pré-requisitos

Antes de começar, garanta que você tem o seguinte software instalado:
- Python (versão 3.8 ou superior)

### Passo 2: Preparar a Pasta do Projeto

1. Descompacte o arquivo .zip do projeto em uma pasta de sua preferência.
2. Abra um terminal (PowerShell, CMD, Terminal, etc.).
3. Navegue com o comando `cd` até a pasta que foi criada. Por exemplo:
   cd C:\Users\SeuUsuario\Downloads\Trabalho-IA-Zelda-Pathfinding

### Passo 3: Configurar o Ambiente Virtual (venv)

É uma boa prática isolar as dependências do projeto para evitar conflitos. Todos os comandos a seguir devem ser executados dentro da pasta do projeto.

a) Crie o ambiente virtual:
   python -m venv .venv

b) Ative o ambiente virtual:
   - No Windows (PowerShell):
     .venv\Scripts\Activate.ps1
   
   - No Mac ou Linux:
     source .venv/bin/activate

Após ativar, você deverá ver um (.venv) no início do seu prompt do terminal.

### Passo 4: Instalar as Dependências

Com o ambiente virtual ativo, instale a única biblioteca necessária (Pillow) usando o arquivo requirements.txt.

pip install -r requirements.txt

### Passo 5: Executar o Programa

Agora, com tudo configurado, execute o programa principal:

python -m src.main
